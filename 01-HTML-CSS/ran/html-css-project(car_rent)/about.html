אודותינו

היסטוריה

חברת "דרכים פתוחות" הוקמה בשנת 1995 על ידי דנה ואמיר, שני חובבי רכבים עם חזון להפוך את חווית השכרת הרכב לפשוטה, נגישה ואמינה. מאז הקמתה, החברה הרחיבה את מגוון הרכבים בצי והפכה לאחת המובילות בתחום בישראל, עם מספר סניפים ברחבי הארץ.

משימה וערכים

משימתנו היא להציע חווית שכירות רכב איכותית, נוחה ובטוחה לכל אחד ואחת. אנו מחויבים לשקיפות מלאה, שירות לקוחות מעולה ותחזוקה קפדנית של כל רכבים בצי, כדי להבטיח את שביעות רצון הלקוחות שלנו.

הצוות שלנו

אנו גאים בצוות המנוסה והמקצועי שלנו, המורכב ממומחי רכב, יועצי שכירות ונציגי שירות, המחויבים למצוינות ולשביעות רצון הלקוחות. הצוות שלנו תמיד זמין לעזור, לייעץ ולהבטיח שחווית השכירות שלך תהיה חלקה ונעימה ככל הניתן.
